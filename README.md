# RoboFriends a simple React App

DEMO : https://arunrr.github.io/RoboFriends

## Getting Started

### Clone the project

```
git clone https://github.com/arunrr/RoboFriends.git
```

### cd into cloned folder

```
cd RoboFriends
```

### Install the required dependencies for the project

```
npm install
```

### Run the project

```
npm start
```
